# eSportsArenaManager

eSports Arena Manager es una plataforma backend para organizar torneos gamer. Permite
administrar juegos, jugadores, equipos, torneos, inscripciones, partidas, resultados, rankings,
sanciones, premios y notificaciones. El problema que resuelve es coordinar torneos competitivos con
reglas claras, trazabilidad y actualización consistente de resultados.
El caso está diseñado para que el equipo construya una solución backend distribuida, evitando una
aplicación monolítica simple. Cada microservicio debe tener responsabilidad propia, su propia base
de datos, su propio CRUD y reglas de negocio que justifiquen su existencia dentro del ecosistema.
